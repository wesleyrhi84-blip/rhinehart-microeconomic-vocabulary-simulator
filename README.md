# Rhinehart Microeconomic Vocabulary Simulator

A Python-based microeconomics vocabulary quiz that randomly generates 10 questions based on important economic terms and definitions.

## Features

* Generates 10 random questions
* Tests knowledge of microeconomic terms and definitions
* Tracks correct and incorrect answers
* Calculates the final quiz score

## Purpose

This project combines Python programming with microeconomics coursework to create an interactive study tool for practicing key economic vocabulary.
